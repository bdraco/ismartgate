Installation¶

Stable release¶

To install ismartgate, run this command in your terminal:

$ pip install ismartgate

This is the preferred method to install ismartgate, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for ismartgate can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/bdraco/ismartgate

Or download the tarball:

$ curl  -OL https://github.com/bdraco/ismartgate/tarball/main

Once you have a copy of the source, you can install it with:

$ python setup.py install
